Nitrokey App 2
==============
Nitrokey App 2 is the graphical application for Nitrokey 3 devices. For Nitrokey Pro and Nitrokey Storage, use `Nitrokey App 1 <https://www.nitrokey.com/download>`__ instead.

Installation
------------
Download it for `Linux <installation_linux>`__ or `Windows <installation_windows>`__.

Supported Features
------------------

- Updating the firmware
- One-Time Passwords (OTP)
- Password Storage

Planned Features
----------------

- PIN management
- macOS support
